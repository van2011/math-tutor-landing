Math Tutor Landing Page (Demo Project)

Одностраничный лендинг для репетитора по математике.
Проект выполнен для портфолио: адаптивный дизайн, аккуратная типографика, чистая структура, полностью сверстан вручную с помощью HTML + CSS.

Используемые технологии:
◉HTML5
◉CSS3 (встроенные стили, без библиотек)
◉Адаптивная вёрстка
◉Малый вес страницы (без фреймворков)

Функциональность:
◉Полностью адаптивный дизайн (mobile-first)
◉Структурированная подача контента
◉Форма заявки с обработкой на стороне клиента
◉Плавная навигация по секциям
◉Лёгкие UI-эффекты без сторонних библиотек

Цель проекта:
Создать современный промо-сайт для репетитора по математике: показать услуги, опыт, программы, стоимость и позволить пользователю быстро оставить заявку.
